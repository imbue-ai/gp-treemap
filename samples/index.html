<gp-treemap> samples

Drop-in Custom Element that paints a "raised tile" treemap onto a single Canvas, pixel-by-pixel. Open any sample below.

Each sample loads ../dist/gp-treemap.bundle.js via a plain <script src> tag, so it runs under file:// or any static server — no modules, no CORS, no bundler.

  • Filesystem — categorical colors by file kind, bytes formatter, default palette.
  • Budget — quantitative colors by variance-vs-plan, diverging palette.
  • Deep hierarchy — ~960 leaves, colored by tree depth.
  • Interactions — hover/click/dblclick/zoom demo with event log.
  • Gradient intensities — 0 / 0.5 / 1 side-by-side.
  • minCellArea threshold — shows small-cell pruning.
  • Located nodes — highlight ring demo.

Source of truth: src/*.js (ES modules). Rebuild the bundle with node tools/build.js. Tests in tests/ generate PNGs into tests/screenshots/.
